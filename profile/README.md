# 🧑🏼‍💻How To Start
### 1. Clone Repository
```bash
$ git clone https://github.com/Techeer-log/Techeer-log.git
```
### 2. env setting in the Settings folder

#### /backend-secret.env
```bash
ACCESS_KEY=
SECRET_KEY=
BUCKET=
REGION=
JWT_TOKEN=
SERVER_PORT=
ACTIVE_PROFILE=
SALT_VALUE=
```
#### /db/db.env
```bash
# MySQL
MYSQL_DATABASE=
MYSQL_ROOT_USER=
MYSQL_ROOT_PASSWORD=
MYSQL_PASSWORD=
```
#### /spring-source.env
```bash
# MySQL
WAIT_HOSTS=
SPRING_DATASOURCE_URL=
SPRING_DATASOURCE_USERNAME=
SPRING_DATASOURCE_PASSWORD=
```

### 3. docker-compose 실행

#### Techeer-log/
```bash
$ 
```
